## 📋 Sobre o Projeto
O **ConectaÊ** é uma plataforma centralizadora (Hub) desenvolvida como parte das Práticas Extensionistas do curso de Sistemas de Informação. O objetivo principal é mitigar a falta de visibilidade de ONGs locais e facilitar o engajamento comunitário, conectando cidadãos a causas sociais de forma intuitiva e acessível.

### 🚀 Funcionalidades Principais
- **Catálogo de Projetos:** Visualização detalhada de ONGs e causas sociais por categoria.
- **Gestão de Voluntariado:** Fluxo para usuários se candidatarem a vagas de apoio em projetos específicos.
- **Agenda de Eventos:** Mural atualizado com eventos promovidos pelas entidades parceiras.
- **Sistema de Apoio:** Opções para suporte direto a eventos e iniciativas sociais.
- **Geolocalização:** Facilidade para encontrar ações próximas ao usuário.

## 🛠️ Tecnologias Utilizadas
- **Frontend:** HTML5, CSS3
- **Design & Prototipagem:** [Figma](https://www.figma.com/design/DUfTL5Nt2TfxwwueNEzo7Y/Contata%C3%AA)
- **Modelagem de Negócio:** Lean Canvas

## ♿ Acessibilidade e Impacto Social
O projeto foi concebido sob as diretrizes do **WCAG 2.1**, garantindo:
- Alto contraste para melhor legibilidade.
- Navegação estruturada para leitores de tela.
- Design responsivo para acesso via dispositivos móveis em comunidades diversas.
